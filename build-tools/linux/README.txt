** STEPS TO INSTALL AND USE **
******************************

* Extract the zip to any place you want. You have already done that.
* If Python 3.5+ is not installed on your system, install it.
  For Debian/Ubuntu : sudo apt-get install python3.5
  For Fedora        : sudo dnf install python3.5

* Also, install pip3 if it is not installed.
* Double click on "install.sh" file and run it in terminal.
* The installer will do the rest of the job.

* The software is installed at ~/21Lane/. 
  You can search it in the Applications Menu.

* Exchange URL : You can find it on your exchange's website.
